# AI 이미지 프롬프트 생성기

사진을 업로드하고 프롬프트를 생성한 뒤, OpenAI API를 이용해 이미지를 생성하는 예제입니다.
사진은 서버에 파일로 저장하지 않고 메모리에서만 처리합니다.

## 중요

- API 키는 절대 `index.html`에 넣지 마세요.
- API 키는 `.env` 파일에만 넣으세요.
- 이전에 채팅에 붙여넣은 API 키는 반드시 삭제하고 새 키를 발급하세요.

## 실행 방법

### 1. 압축 풀기

`ai-prompt-generator.zip` 파일을 원하는 폴더에 압축 해제합니다.

### 2. `.env` 파일 만들기

`.env.example` 파일을 복사해서 이름을 `.env`로 바꿉니다.

그 다음 `.env` 안에 새로 발급한 OpenAI API 키를 넣습니다.

```env
OPENAI_API_KEY=sk-proj-새로발급한키
```

### 3. 설치

Windows 기준: 폴더 주소창에 `cmd`를 입력하고 Enter를 누른 뒤 아래 명령어를 실행합니다.

```bash
npm install
```

### 4. 서버 실행

```bash
npm start
```

성공하면 아래처럼 보입니다.

```text
서버 실행 중: http://localhost:3000
```

### 5. 웹페이지 열기

브라우저 주소창에 아래 주소를 입력합니다.

```text
http://localhost:3000
```

`index.html`을 더블클릭해서 열면 API가 작동하지 않습니다.
반드시 `npm start` 후 `http://localhost:3000`으로 접속해야 합니다.

## 사용 방법

1. 사진을 업로드합니다.
2. 스타일, 색상, 분위기, 구도를 선택합니다.
3. `프롬프트 만들기`를 누릅니다.
4. `GPT API로 이미지 생성`을 누릅니다.

## 테스트 모드

테스트 모드를 켜면 실제 API를 호출하지 않고 업로드한 사진을 결과 영역에 표시합니다.
웹페이지 동작만 확인할 때 사용하세요.

## API 키 위치

API 키는 여기에만 들어갑니다.

```text
.env
```

절대 아래 위치에 넣지 마세요.

```text
index.html
server.js 안의 문자열
채팅창
공유 문서
```
